myFlix Movie Directory

REST API with Node.js

Server side movie web application

Users will have access information about a movie's description, genre and director

Once registered, users will be able to update their information and create a list of favorite movies

Documentation
Request URL HTTP Method
Return a list of ALL movies to the user /movies GET
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/title GET
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/title GET
Return data about a director (bio, birth year, death year) by name /movies/directors/name GET
Allow new users to register /users POST
Allow users to update their user info (username) /users/username PUT
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/username/movies/movieName POST
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/username/movies/movieName DELETE
Allow existing users to deregister (showing only a text that a user email has been removed) /users/username DELETE
